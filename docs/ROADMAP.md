# ROADMAP

## Q1
- 发布基础 PaddleOCR Server：`POST /ocr`、Docker 镜像（latest/cuda12.6）、模型缓存挂载指南
- 建立最小监控与日志输出，支持健康检查
- 新增轻量 Web GUI（/ui）：单页原生 HTML/JS，拖拽/选择图片，调用 /ocr 并展示结果；与 REST API 共存于同一镜像

## Q2
- 接入认证/限流与基本安全配置，提供示例网关/反向代理配置
- 暴露 Prometheus/OpenTelemetry 指标，完善报警基线
- 轻量 Web GUI 增强：复制/导出/历史/批处理体验完善，元信息展示与参数面板（不引入重框架）

## Q3
- 扩展多端点：版面/表格识别、批处理与队列化调度
- 支持多模型/多语言热切换与按需加载，降低冷启动
- GUI 进阶：多图结果导出打包、基本主题与可达性优化（可选）

## Q4
- 提供 Helm/Compose 一键部署包，覆盖私有云与边缘场景
- 弹性伸缩与成本优化策略（自动扩缩容、模型缓存共享）
